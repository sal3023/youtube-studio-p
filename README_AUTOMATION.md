# 🚀 YouTube Studio Pro (Automated Workflow)
## نظام أتمتة متكامل لقناة الصحة واللياقة البدنية - إصدار 2026

مرحباً بك في **YouTube Studio Pro**، النظام البرمجي الأول من نوعه لإدارة قناتك على يوتيوب بشكل آلي تماماً باستخدام GitHub Actions والذكاء الاصطناعي. تم تصميم هذا المستودع ليعمل كمحرك خلفي (Backend) لقناتك، حيث يقوم بتوليد الأفكار، كتابة السيناريوهات، وتصميم مفاهيم الصور المصغرة تلقائياً.

---

### 🛠️ هيكل النظام (System Architecture)

تم تنظيم المستودع لضمان سهولة الوصول والأتمتة الكاملة:

| المجلد | الوظيفة |
| :--- | :--- |
| `.github/workflows/` | يحتوي على ملفات الأتمتة التي تشغل السكربتات بجدول زمني. |
| `scripts/` | يحتوي على محركات الذكاء الاصطناعي (Python) لتوليد المحتوى. |
| `content/ideas/` | يتم تخزين أفكار الفيديوهات المولدة آلياً هنا. |
| `content/scripts/` | يتم تخزين السيناريوهات الكاملة هنا. |
| `content/thumbnails/` | يحتوي على مفاهيم الصور المصغرة وأوامر (Prompts) التصميم. |
| `content/videos/` | يحتوي على بيانات السيو (SEO) والعناوين والأوصاف. |

---

### 🤖 كيف يعمل النظام آلياً؟

يعمل النظام بناءً على جدول زمني محدد مسبقاً (Cron Jobs) لضمان استمرارية القناة دون تدخل بشري دائم:

1.  **كل يوم اثنين**: يتم تشغيل `generate_content.yml` لتوليد فكرة فيديو جديدة وسيناريو كامل.
2.  **كل يوم ثلاثاء**: يتم تشغيل `generate_thumbnails.yml` لتوليد 3 مفاهيم مختلفة للصور المصغرة بناءً على سيناريو يوم الاثنين.
3.  **عند الطلب**: يمكنك تشغيل أي سير عمل يدوياً من تبويب **Actions** في GitHub.

---

### 🚀 كيف تبدأ الاستخدام؟

لربط قناتك وتفعيل الأتمتة، اتبع الخطوات البسيطة التالية:

1.  **إضافة مفاتيح API**: اذهب إلى إعدادات المستودع (Settings) -> Secrets and variables -> Actions، وأضف الأسرار التالية:
    *   `OPENAI_API_KEY`: مفتاح API الخاص بك (يمكن استخدامه مع Groq أو OpenRouter).
    *   `LLM_API_BASE`: رابط API الأساسي للنموذج الذي ستستخدمه (مثال: `https://api.groq.com/openai/v1` لـ Groq، أو `https://openrouter.ai/api/v1` لـ OpenRouter).
    *   `LLM_MODEL`: اسم النموذج الذي ستستخدمه (مثال: `llama3-8b-8192` لـ Groq، أو `mistralai/mistral-7b-instruct` لـ OpenRouter).
2.  **تفعيل GitHub Actions**: تأكد من أن الـ Actions مفعلة في مستودعك.
3.  **تخصيص السكربتات**: يمكنك تعديل ملفات Python في مجلد `scripts/` لتغيير نبرة الصوت أو نوع المحتوى (مثلاً من "الصحة" إلى "التقنية").

---

### ⚙️ إعدادات النماذج اللغوية الكبيرة (LLM Configuration)

يمكنك التحكم في النموذج اللغوي الكبير (LLM) الذي يستخدمه النظام عن طريق ضبط متغيرات البيئة في GitHub Secrets:

| المتغير | الوصف | أمثلة للقيم (Groq) | أمثلة للقيم (OpenRouter) |
| :--- | :--- | :--- | :--- |
| `OPENAI_API_KEY` | مفتاح API الخاص بك (يستخدمه Groq و OpenRouter). | `gsk_xxxxxxxxxxxxxxxx` | `sk-or-xxxxxxxxxxxxxxxx` |
| `LLM_API_BASE` | رابط API الأساسي. | `https://api.groq.com/openai/v1` | `https://openrouter.ai/api/v1` |
| `LLM_MODEL` | اسم النموذج اللغوي. | `llama3-8b-8192` | `mistralai/mistral-7b-instruct` |

**ملاحظة**: يمكنك العثور على أحدث النماذج والروابط في وثائق Groq و OpenRouter الرسمية.
